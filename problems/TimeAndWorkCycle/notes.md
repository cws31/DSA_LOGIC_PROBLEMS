# Notes for Time and Work Cycle

- Calculate rates: Ananya rate = 1/18 per day, Bharat rate = 1/27 per day.
- A 3-day cycle consists of:
  - Day 1: Ananya works (1/18)
  - Day 2: Ananya works (1/18)
  - Day 3: Bharat works (1/27)
- Total work done in 1 cycle (3 days) = 2/18 + 1/27 = 4/27.
- Complete 6 cycles (18 days): 24/27 work done. Remaining: 3/27 = 1/9.
- Day 19 (Ananya works): 1/18 work done. Remaining: 1/9 - 1/18 = 1/18.
- Day 20 (Ananya works): 1/18 work done. Assignment complete!
- Total time: 20 days.
